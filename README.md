# AI Mock Interview Platform

An AI-powered interview preparation platform that helps users practice technical and HR interviews using Generative AI.

## Features (Planned)

- User Authentication
- Resume Upload
- AI Interview Question Generation
- Voice-based Interviews
- AI Feedback & Scoring
- Interview History
- Dashboard
- PostgreSQL Database
- React Frontend
- FastAPI Backend

## Tech Stack

- React
- FastAPI
- PostgreSQL
- Gemini API
- Python
